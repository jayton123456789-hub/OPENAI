# Veilbound Art Direction

## Visual identity

Veilbound combines dark botanical watercolor, antique paper, restrained gold filigree, translucent forest fog, and original ceremonial masks. It should feel secretive and premium, never horrific or casino-like.

## Palette

| Token | Hex | Purpose |
| --- | --- | --- |
| Forest Ink | `#161F16` | Primary field and shadows |
| Deep Moss | `#3A442E` | Panels and elevated surfaces |
| Sage Bronze | `#6B6749` | Secondary controls and borders |
| Muted Gold | `#A49168` | Ornament and inactive emphasis |
| Antique Gold | `#C5AA78` | Primary actions and titles |
| Parchment Ivory | `#DFCDAD` | Cards, messages, and main copy |
| Veil Glow | `#B8D287` | Active-turn and success highlight only |

## Production asset policy

- Card typography, borders, corner marks, and layout are code-rendered for perfect consistency.
- Generated art supplies expressive portraits, botanical backgrounds, and the card-back illustration.
- Runtime images are optimized WebP files; original source generations are retained separately.
- One consistent reference atlas drives all thirteen portraits.
- Masks must remain original and must not duplicate the user’s mood-reference artwork.
- Avoid text inside generated images, excessive glow, realistic gore, poker imagery, and tiny decorative details that disappear on mobile.

## Runtime assets

- One portrait botanical environment
- One symmetrical 5:7 card back
- Thirteen square identity portraits
- Code-native crest, borders, badges, and interaction ornament

## Motion

Movement is slow and deliberate: soft lifts, reveals, pulsing deck light, and curtain handoffs. Reduced-motion mode removes all nonessential animation while preserving state feedback.
